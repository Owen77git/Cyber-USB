# Cyber USB Toolkit

## Overview

Cyber USB Toolkit is a versatile bootable USB utility that is used in system performance optimization and cybersecurity protection. It offers an all round set of tools to both Windows as well as Linux system via a simple interface.

## Features

### System Performance
Cleaning up of system and removal of temporary files.
Managerial information and updates on drivers.
Disk health monitoring
Performance optimization
Power (laptops) management.

### System Security
Antivirus and viruses screening.
Firewall set up verification.
Credential audit Password audit.
Network security scanning
Phishing prevention and education.
Education Ethical hacking tools (only education)

## Quick Start

### Requirements
USB drive (not less than 8GB, preferably 16GB)
Windows 10/11, system or Linux or Kali Linux.
Full-function administrative/root privileges.
